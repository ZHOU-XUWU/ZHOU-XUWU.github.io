Welcome to my personal site, this website use hexo structure and deployed on GitHub. Detail steps as following.

Github - New repository

According the instructions to creating your repository, a repository contains all the files for your project. Repository name should end with '@github.io‘

Generate key for your repository

Open local computer terminal and type in 

$ ssh-keygen -t rsa -C "your github account email address"

After the key generated, you will get two file — “id_rsa” and “id_rsa.pub”. Open the “id_rsa.pub”, copy the whole content in the file and paste them in https://github.com/settings/ssh

Click the Button “New SSH Key” than paste the content of “id_rsa.pub” into key, full the title as your wish.

This step make your local conputer authoritative to control(deploy) your project.

Install node.js & Git

Install by downloading from offical website.

node.js: https://nodejs.org/en/ (at least node.js 6.9)

Git: http://git-scm.com/downloads (or brew install git)

Install hexo

Create a new folder for storing your website files, eg. Blog

$ cd Blog
Blog $ npm install hexo-cli -g

If any question happen when you install hexo, please check whether your macOS has installed Xcode or not. If the answer is not, please install Xcode first.

Type hexo in your terminal to check whether the hexo has been installed in your computer

$ hexo

it should have correct responses.

Deploy 

// initialize your website and keep them in a new folder
Blog $ hexo init <FolderName>
Blog $ cd FolderName
//install all dependencies 
FolderName $ npm install

Then find the _config.yml in your new folder and edit it.

Edit your webiste info 

Title:
SubTitle:
Description: write for search engine
Author:
Language:
Timezone:

PS: Special constraints for Language and timezone, if you don’t know how to deal with it, check this Langusge and timezone.

Deploy your project 

deploy:
    type: git
    repo: https://github.com/GithubAccountName/repoName.github.io.git
    branch: master

Save then close _config.yml

install hexo deploy iin terminal

$ npm install hexo-deployer-git --save
$ hexo deploy repoName.giithub.io
